# iPod Music Player - 高保真原型

基于经典iPod设计的现代音乐播放器原型集合，完美还原经典iPod体验，同时集成Apple Music服务。

## 🎯 项目概述

这是一个完整的iPod风格音乐播放器原型，包含：
- 经典iPod 60/40黄金比例布局
- 完整的转盘交互设计
- Apple Music服务集成
- 用户友好的权限和错误处理
- 响应式设计和现代UI

## 📁 项目结构

```
prototype/
├── index.html                 # 主入口页面
├── README.md                  # 项目说明文档
├── public/                    # 公共资源
│   ├── css/
│   │   └── styles.css         # 共享样式文件
│   ├── js/
│   │   └── interactions.js    # 交互逻辑
│   └── assets/
│       └── icons/             # 图标资源
└── app/                       # 原型页面
    ├── main-player.html       # 主播放界面
    ├── wheel-control.html     # 转盘控制组件
    ├── menu-navigation.html   # 菜单导航界面
    ├── music-library.html     # 音乐库浏览界面
    ├── permission-request.html # 权限申请界面
    └── error-states.html      # 错误状态界面
```

## 🚀 快速开始

1. **直接访问**：打开 `index.html` 文件查看完整原型集合
2. **独立页面**：访问 `app/` 目录下的任意 HTML 文件查看单独原型
3. **本地服务器**：建议使用本地服务器运行以获得最佳体验


## 🎨 设计特色

### 核心界面设计
- **屏幕区域 (60%)**：专辑封面、歌曲信息、播放进度
- **转盘区域 (40%)**：经典iPod转盘，支持完整交互
- **色彩方案**：白色主体 + 深灰辅助 + Apple蓝强调色

## 🔄 交互规则

### 转盘操作
- **慢速旋转** (< 5 rad/s)：音量调节，每30度 = 1级
- **快速旋转** (> 5 rad/s)：歌曲切换，连续90度触发
- **长按旋转** (> 500ms)：播放进度控制

### 按钮功能
- **中央按钮**：单击播放/暂停，长按菜单
- **上键**：菜单/音量+
- **下键**：播放/暂停
- **左键**：上一首
- **右键**：下一首

### 手势支持
- **点击**：选择和确认
- **长按**：上下文菜单
- **旋转**：音量和进度控制
- **滑动**：页面切换
